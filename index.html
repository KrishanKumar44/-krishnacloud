Linux

Q1. What is Soft Link?

Answer

Soft Link (Symbolic Link) is a shortcut to another file or directory. If original file is deleted, Soft Link becomes broken.

Command


ln -s source_file softlink

Real Time Example

Apache and Nginx use symbolic links for website configuration.
