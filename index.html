📋 Basic Interview Questions & Answers

Linux

Q1: What is the difference between Soft Link and Hard Link?

Answer: A Soft Link (Symlink) is a shortcut to the original file path. If the original file is deleted, the soft link breaks. A Hard Link is a direct pointer to the file's data (same Inode number). Even if the original file is deleted, you can still access the data through the hard link.

Cloud

Q2: What is Cloud Computing and its service models?

Answer: Cloud computing is the delivery of computing services (servers, storage, databases, networking) over the internet. There are three main service models:

• IaaS (Infrastructure as a Service): Gives you raw infrastructure (e.g., AWS EC2, Azure VM).
• PaaS (Platform as a Service): Gives you a platform to deploy code without managing servers (e.g., AWS Elastic Beanstalk, GitHub Pages).
• SaaS (Software as a Service): Fully managed ready-to-use software (e.g., Gmail, Google Drive).

Linux

Q3: How do you check RAM and CPU usage in Linux?

Answer: To check RAM usage, we use the free -m or free -h command. To check CPU usage and live running processes, we use the top or htop command.
